üìêFormulas Used

DC Resistance: $$R_{\mathrm{DC}} = \frac{\ell}{\pi\,r_{0}^{2}\,\sigma}$$

AC Resistance: $$R_{\mathrm{total}} = (1 + G_{p})\,R_{\mathrm{skin}}$$, where \(G_p\) is the proximity factor.

Self-Inductance: $$L_{\mathrm{self}} = \sum_{i=1}^{N}\sum_{j=1}^{N} M_{ij}$$

where \(M_{ij}\) is the mutual inductance between turns i and j, calculated using elliptic integrals K(m) and E(m).

The elliptic integral code is based on the implementation by duetosymmetry (Leo C. Stein).
